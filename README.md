# UI-UX Repository

This repository contains UI/UX-focused experiments and interactive frontend examples.

## Projects

| Project | Description | Stack |
| --- | --- | --- |
| `mortgage-calculator` | Interactive mortgage calculator with affordability guidance and animated UI | React, TypeScript, Vite, GSAP, CSS |

## Getting Started

From this repository root:

```bash
cd mortgage-calculator
npm install
npm run dev
```

## Scripts (mortgage-calculator)

```bash
npm run dev      # start local dev server
npm run build    # create production build
npm run preview  # preview production build locally
```

- Each project folder can have its own README with implementation details.
